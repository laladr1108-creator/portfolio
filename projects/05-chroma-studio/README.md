# Chroma Studio

Креативный редактор палитр для дизайнеров и бренд-систем.

## Возможности

- Генерация гармоничных палитр от базового оттенка.
- Переключение режима гармонии и интенсивности.
- Копирование отдельных HEX-значений.
- Сохранение палитр в рамках текущей сессии.
- Проверка контраста между фоном и текстом.

Откройте `index.html` напрямую или через главную страницу портфолио.
