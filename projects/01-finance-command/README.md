# Финансовый центр

Премиальный SaaS-дашборд для основателей, руководителей и операционных команд.

## Возможности

- Обзор KPI: выручка, burn rate, runway и валовая маржа.
- Интерактивные вкладки метрик с адаптивным SVG-графиком.
- Сценарные слайдеры для роста и темпа найма.
- Таблица операций с поиском и статусами риска.
- Поддержка reduced motion и мобильная раскладка.

Откройте `index.html` напрямую или через главную страницу портфолио.
