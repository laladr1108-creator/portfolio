# Nova Travel

Интерактивный планировщик премиальных путешествий и маршрутов.

## Возможности

- Карточки направлений с фотографиями и метаданными поездки.
- Динамические точки на карте и сводка маршрута.
- Слайдер бюджета с живым расчетом.
- Дневной план с добавлением и удалением остановок.
- Адаптивная верстка, подходящая для GitHub Pages.

Откройте `index.html` напрямую или через главную страницу портфолио.
