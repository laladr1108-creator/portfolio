# Atelier Commerce

Премиальный e-commerce каталог для curated lifestyle товаров.

## Возможности

- Сетка товаров с реальными изображениями.
- Фильтры по категории и визуальному тону.
- Сортировка по подборке, цене и рейтингу.
- Выезжающая корзина с количеством, итогом и пустым состоянием.
- Адаптивный премиальный интерфейс магазина.

Откройте `index.html` напрямую или через главную страницу портфолио.
