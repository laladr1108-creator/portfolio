# AI-планировщик спринта

Операционный интерфейс для продуктовой команды и планирования спринтов.

## Возможности

- Фильтрация бэклога по приоритету.
- Доска спринта с планом работ и индикатором загрузки.
- Добавление задач из бэклога в текущий спринт.
- Генерация короткого AI-брифа для команды.
- Адаптивная верстка, аккуратные состояния и анимация.

Откройте `index.html` напрямую или через главную страницу портфолио.
