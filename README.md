# Портфолио Swatik11

Статическое frontend-портфолио Андрея. Главная страница ведет на 5 рабочих проектов:

1. **КБЖУ Coach** - дневник питания с макросами, пресетами и прогрессом.
2. **VPN Guard** - личный кабинет VPN-сервиса с серверами, статусом и устройствами.
3. **Exclusive Partners** - CRM-панель для партнеров, фильтры и расчет комиссии.
4. **WorkBoard** - мини-доска задач со статусами и добавлением задач.
5. **Money Map** - трекер расходов, бюджет и прогресс месяца.

Все работает без сборки: HTML, CSS и JavaScript.

## Локальный запуск

```powershell
python -m http.server 5173
```

Открыть:

```text
http://127.0.0.1:5173/
```
