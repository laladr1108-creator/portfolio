# Портфолио Swatik11

Статическое портфолио Андрея: frontend-интерфейсы, FluxVpn и Telegram-боты.

## Проекты

1. **КБЖУ Bot** - Telegram-бот и мини-интерфейс для рациона, макросов и прогресса.
2. **FluxVpn** - личный кабинет VPN-сервиса с серверами, тарифом и устройствами.
3. **Exclusive Partners** - CRM-панель для партнеров, фильтры и расчет комиссии.
4. **Telegram Order Bot** - бот заказов с меню, корзиной и статусами заявки.
5. **Telegram Support Bot** - бот поддержки с заявками, приоритетами и шаблонами ответов.

Все работает без сборки: HTML, CSS и JavaScript.

## Локальный запуск

```powershell
python -m http.server 5173
```

Открыть:

```text
http://127.0.0.1:5173/
```
