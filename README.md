# Премиум Frontend Портфолио

Папка содержит 7 самостоятельных frontend-проектов для портфолио. Все проекты работают без сборки: можно открыть `index.html` или запустить локальный статический сервер.

## Проекты

1. **Финансовый центр** - SaaS-дашборд для финансовых метрик, сценариев, графиков и таблицы операций.
2. **AI-планировщик спринта** - инструмент для приоритизации задач, планирования спринта и генерации краткого брифа.
3. **Atelier Commerce** - премиальный интернет-магазин с фильтрами, сортировкой, карточками товаров и корзиной.
4. **Nova Travel** - интерактивный планировщик путешествий с маршрутами, бюджетом и дневным планом.
5. **Chroma Studio** - редактор цветовых палитр с генератором, сохранением, копированием HEX и проверкой контраста.
6. **КБЖУ Planner** - калькулятор дневной нормы, макросов, приемов пищи и прогресса по цели.
7. **Exclusive Partners** - партнерская CRM-панель с карточками, фильтрами, pipeline и расчетом комиссии.

## Как открыть

Откройте файл `index.html` в корне папки или запустите сервер:

```powershell
python -m http.server 5173
```

После этого откройте:

```text
http://localhost:5173
```

## Публикация на GitHub

Самый простой путь после установки Git и GitHub CLI:

```powershell
.\publish-to-github.ps1
```

Если нужно вручную:

```powershell
git init
git add .
git commit -m "Добавить премиум портфолио"
gh repo create premium-frontend-portfolio --public --source=. --remote=origin --push
```

Для GitHub Pages включите Pages в настройках репозитория: `Settings -> Pages -> Deploy from a branch -> main -> /root`.
